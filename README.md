# FreeLanceHub Backend

This is the Spring Boot backend for the FreeLanceHub freelance management system.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JWT
- MongoDB
- Maven

## Main Modules
- Authentication
- Users
- Jobs
- Proposals
- Contracts
- Payments
- Admin management

## Run Locally
mvn clean install
mvn spring-boot:run
